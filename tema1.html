Minería de Datos de Excel

Definición

La minería de datos de Excel es un tipo de análisis de datos que trata de descubrir patrones, tendencias y relaciones que puedan ayudar en algún aspecto.

Tipos de análisis:

  • Preparación de Datos: Limpieza, filtrado y ordenado.
  • Análisis Exploratorio: Uso de tablas y gráficas.
  • Herramientas de análisis: Uso de herramientas.
  • Funciones y Fórmulas: Uso de funciones como BUSCARV.
