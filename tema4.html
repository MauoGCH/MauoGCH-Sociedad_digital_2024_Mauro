Hardware

Definición

El hardware consiste en todas las partes físicas y tangibles del sistema. Es decir, todos los componentes que se pueden tocar.

Ejemplos de Hardware

  • CPU
  • Memoria:
    • RAM
    • ROM
  • Almacenamiento:
    • Discos Duros (HDD)
    • Unidades Sólidas (SSD)
  • Unidades Ópticas:
    • CD
    • DVD
  • Periféricos de entrada
  • Placa Base (Motherboard)
  • Tarjetas de Expansión
  • Fuentes de Poder
